How to write a Tezos protocol
=============================

A Tezos node is parameterized by a software component called an
*economic protocol* (or protocol for short). Different protocol
implementations can be used to implement different types of blockchains.

This is a two-part tutorial on how to implement such a protocol, based on two demo protocols that are embedded in the Octez node.
